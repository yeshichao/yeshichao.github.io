基于hugo部署到github搭建个人博客

1. Hugo是什么？官网链接

Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。 Hugo还提供有Hugo中文文档，文档里面也有介绍如何安装Hugo到将个人站点部署到Github仓库上。本文基于ubuntu+git下操作

2. 如何安装Hugo

  1. 如果你是macOs 用户可以使用homebrew 来安装：
brew install hugo
  1. 如果你是Ubuntu 或者其他linux 用户可以使用apt 来安装：
sudo apt-get install hugo
  1. 如果是Windows 用户，使用scoop 快速安装：
scoop install hugo

使用包管理工具，基本上可以实现一条代码快速安装，具体可以参考Hugo官网

这里我在实操的时候遇到一个问题，我通过apt安装的hugo不是最新版本的。而我选的主题要求hugo版本比较高。可以通过 hugo version来查看自己当前hugo版本。所以后来通过去github上hugo的仓库，在releases里下载最新版本的压缩包。

3. 创建站点

自己选择合适的文件夹下使用终端输入命令：

hugo new site myblog

此命令会在当前文件夹下建立一个名叫myblog的文件夹，里面有很多子文件夹(content,themes等等)；

接下来就是选择你喜爱的主题hugo主题可以随意挑一个自己喜爱的主题使用，也可以在github中搜索你想要的主题库，使用命令：

cd themes
git submodule add https://github.com/hossainemruz/toha.git themes/toha

我这里选用了toha主题。

然后可以通过 hugo server -t 主题名称 --buildDrafts预览主题。通过返回的本机链接，用浏览器打开就可以看到个人博客当前主题的样子。

4. 第一篇博客

myblog/content文件夹是存放你博客文章的地方，在此新建.md文件。回到myblog根目录：

hugo new posts/XXXX.md

写好内容之后，再在本机预览一遍，输入预览命令hugo server -t 主题名称 --buildDrafts，打开链接即可预览； 预览成功后，接下来便是把整个站点推到Github仓库中。

5. 新建github仓库

在Github主页选择New rpository选项： 这里注意Owner与Repository name尽量要保持一致，并在后面加上.github.io

创建完之后就是把站点部署到这个仓库中了。

6. 部署到Github

  1. 关联你的Github：输入hugo --theme=主题名称 --baseUrl="https://xxx.github.io"(xxx表示你的github..) --buildDrafts 按回车键后发现根目录下多出一个public的文件夹；
  2. 输入cd public切换到public目录下编辑： 一行一行输入
git init
git add .
git commit -m "commit message"
git remote add origin https://github.com/xxx/xxx.github.io.git 
git push -u origin master

如果第二次上传需要修改的内容： 在执行git push -u origin master之前先执行git pull origin master 最后访问xxxxx.github.io这个网站，便是你的博客。至此静态博客Hugo完成。

7. 配置主题

每个主题的配置方法都不同，一般在其网站上会有写。Toha主题的配置方法在这，具体内容在这。接下来节将一些我在配置中的问题。

7.1 Hudo上数学公式现实问题

这是官方的方法,使用了KaTex。意思是先在主题的/layouts/partials/下创建math.html。然后复制该网址中提供的，

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.11.1/dist/katex.min.css" integrity="sha384-zB1R0rpPzHqg7Kpt0Aljp8JPLqbXI3bhnPWROx27a9N0Ll6ZP/+DiW/UqRcLbRjq" crossorigin="anonymous">
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.11.1/dist/katex.min.js" integrity="sha384-y23I5Q6l+B6vatafAwxRu/0oK/79VlbSz7Q9aiSZUvyWYIYsd+qj+o24G5ZU2zJz" crossorigin="anonymous"></script>
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.11.1/dist/contrib/auto-render.min.js" integrity="sha384-kWPLUVMOks5AQFrykwIup5lo0m3iMkkHrD0uJ4H5cjeGihAutqP0yW0J6dpFiVkI" crossorigin="anonymous"
    onload="renderMathInElement(document.body);"></script>

黏贴到math.html。

然后打开myblog/archetypes/default.md可添加以下默认选项：

title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
draft: true
math: true

当有 math: true时，数学公式全部用两个美元符号包围起来，就像这样 $$ f(x) = \lim_{n \to \infty}x_n $$ 变可以在行内显示公式 $$ f(x) = \lim_{n \to \infty}x_n $$ 注意：许多博客上的配置语句是用等号，有可能会导致主题文件编译的时候出现错误，建议使用:来分隔

7.2 如何修改文章页面的名字和头像

名字只需打开myblog/archetypes/default.md添加

author:
  name: "xxxx"

即可。

头像需要打开 /home/ysc/myblog/themes/toha/static/assets/images将其中 default-avatar.png改成自己需要的图像，注意格式必须是.png。
